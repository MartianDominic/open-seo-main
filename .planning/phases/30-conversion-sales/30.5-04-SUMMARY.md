## Stage Transition Rules

```
new        -> [analyzing, archived]
analyzing  -> [scored, new]         // can retry on failure
scored     -> [qualified, contacted, archived]
qualified  -> [contacted, archived]
contacted  -> [negotiating, archived]
negotiating -> [converted, contacted, archived]
converted  -> []                    // terminal
archived   -> [new]                 // re-activate
```

## Automation Flow

1. Prospect created -> pipelineStage = "new"
2. Analysis queued -> stays "new" (manual transition or future hook)
3. Analysis completes -> auto-transition to "scored"
4. If priorityScore >= 70 -> auto-transition to "qualified"
5. Manual stages: contacted, negotiating, converted, archived
