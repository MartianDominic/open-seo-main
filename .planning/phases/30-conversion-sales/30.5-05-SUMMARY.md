## Implementation Details

### Server Functions (Task 1)
- `bulkAnalyzeProspects`: Queues analysis for up to 500 prospects, respects 10/day quota
- `bulkArchiveProspects`: Transitions selected prospects to archived stage with ownership verification
- `getStageDistribution`: Returns prospect counts grouped by pipeline stage
- `getRemainingQuota`: Returns remaining daily analysis quota

### ProspectsTable (Task 2)
- TanStack Table v8 with `useReactTable` hook
- Row selection via `enableRowSelection` and `RowSelectionState`
- `IndeterminateCheckbox` component handles three-state header checkbox
- Stage badges with Lithuanian labels and color coding
- Priority score with green (>=70), amber (>=40), gray (<40) bands
- Click isolation: checkbox clicks don't trigger row navigation

### BulkActionBar (Task 3)
- Floating action bar with framer-motion slide-up animation
- Analyze dropdown with Quick Scan, Deep Dive, Opportunity Discovery options
- Archive button with loading state
- Export CSV button (client-side download via buildCsv/downloadCsv)
- Clear selection button
- Quota warning when selected count exceeds remaining quota

### PipelineStageChart (Task 4)
- Recharts horizontal BarChart with ResponsiveContainer
- Color-coded bars per pipeline stage
- Custom tooltip showing stage label and prospect count
- Empty state when no prospects

### Page Integration (Task 5)
- Pipeline distribution chart card above table
- ProspectsTable replaces basic HTML table
- BulkActionBar renders when rows are selected
- Data refresh after bulk actions complete
- Row click navigates to prospect detail page
