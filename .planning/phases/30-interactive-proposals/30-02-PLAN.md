# Plan 30-02: AI Lithuanian Generation

## Goal

Generate proposal content in Lithuanian using Gemini 3.1 Pro with brand voice consistency.

## Key Design Decision

**Segment-by-segment generation** for proposals (structured multi-section content needs consistency).
- Proposals: 6 segments generated sequentially
- Articles: Single-shot generation (Gemini handles well)

   - Error handling and retries

2. **Lithuanian Brand Voice System**
   - Terminology dictionary

   - Per-agency customization

3. **Segment Generator**
   - Hero/summary segment
   - Current state segment
   - Opportunities segment
   - ROI projection segment
   - Investment segment
   - Next steps segment

```typescript
// server/lib/proposals/gemini.ts
import { GoogleGenerativeAI } from "@google/generative-ai";

const genAI = new GoogleGenerativeAI(process.env.GOOGLE_API_KEY!);

const LITHUANIAN_BASE_SYSTEM = `
Esi profesionalus SEO pasiūlymų rašytojas lietuvių kalba.

TONAS:
- Profesionalus, bet šiltas
- Oficialus "Jūs" (ne "tu")
- Konkretūs skaičiai, ne tuščios frazės
- Pasitikintis, bet ne arogantiškas

TERMINOLOGIJA (neversk):
- SEO → SEO
- ROI → ROI
- CPC → CPC
- Google Search Console → Google Search Console

TERMINOLOGIJA (versk):
- Keywords → raktažodžiai
- Organic traffic → organinis srautas
- Conversion rate → konversijos rodiklis
- Domain authority → domeno autoritetas
- Backlinks → nuorodos
- Search volume → paieškų skaičius

VENGTI:
- "Garantuojame rezultatus" (neteisėta)
- "Geriausi specialistai" (tuščia)
- "Unikalūs sprendimai" (tuščia)
- Anglų kalbos terminų be reikalo
`;

type Segment = 'hero' | 'current_state' | 'opportunities' | 'roi' | 'investment' | 'next_steps';

const SEGMENT_PROMPTS: Record<Segment, (data: ProspectAnalysis) => string> = {
  hero: (data) => `
Sugeneruok pasiūlymo antraštę ir paantraštę.

DUOMENYS:
- Domenas: ${data.domain}
- Neišnaudota srauto vertė: €${data.trafficValue}/mėn.
- Galimybių skaičius: ${data.opportunities.length} raktažodžių

FORMATAS (JSON):
{
  "headline": "Pagrindinė antraštė (max 10 žodžių)",
  "subheadline": "Paantraštė su konkrečiu skaičiumi (max 20 žodžių)"
}

Pavyzdys:
{
  "headline": "Jūsų svetainė turi €10,800/mėn. neišnaudotą potencialą",
  "subheadline": "Radome 156 raktažodžius, kuriais galėtumėte pritraukti naujų klientų"
}
`,
  
  current_state: (data) => `
Aprašyk dabartinę svetainės situaciją 2-3 sakiniais.

DUOMENYS:
- Dabartinis srautas: ${data.traffic}/mėn.
- Raktažodžių skaičius: ${data.keywords}
- Dabartinė srauto vertė: €${data.currentValue}/mėn.

FORMATAS: Paprastas tekstas, 2-3 sakiniai. Neutralus tonas, tik faktai.
`,

  opportunities: (data) => `
Aprašyk rastas galimybes 2-3 sakiniais.

DUOMENYS:
- Galimybių skaičius: ${data.opportunities.length}
- Top 3 raktažodžiai: ${data.opportunities.slice(0, 3).map(o => o.keyword).join(', ')}
- Bendra potenciali vertė: €${data.trafficValue}/mėn.

FORMATAS: Paprastas tekstas, 2-3 sakiniai. Įkvepiantis, bet realus tonas.
`,

  roi: (data) => `
Aprašyk investicijų grąžą 2-3 sakiniais.

DUOMENYS:
- Prognozuojamas srauto prieaugis: +${data.projectedTrafficGain}/mėn.
- Srauto vertė: €${data.trafficValue}/mėn.
- Mėnesinė investicija: €${data.monthlyFee}

FORMATAS: Paprastas tekstas, 2-3 sakiniai. Aiškus, konkretus.
`,

  investment: (data) => `
Aprašyk investiciją ir kas įeina į paslaugą.

DUOMENYS:
- Įdiegimo mokestis: €${data.setupFee}
- Mėnesinis mokestis: €${data.monthlyFee}
- Paslaugos: ${data.inclusions.join(', ')}

FORMATAS (JSON):
{
  "description": "1-2 sakiniai apie investiciją",
  "value_proposition": "1 sakinys kodėl verta"
}
`,

  next_steps: (data) => `
Sugeneruok 3 aiškius žingsnius, ką klientas turėtų daryti toliau.

FORMATAS (JSON array):
["Pirmas žingsnis", "Antras žingsnis", "Trečias žingsnis"]

Pavyzdys:
["Peržiūrėkite pasiūlymą", "Pasirašykite sutartį elektroniniu parašu", "Pradėsime per 48 valandas"]
`
};

export async function generateProposalSegment(
  segment: Segment,
  data: ProspectAnalysis,
  brandVoice?: BrandVoiceConfig
): Promise<string> {
  const model = genAI.getGenerativeModel({ 
    model: "gemini-1.5-pro",
    systemInstruction: LITHUANIAN_BASE_SYSTEM + (brandVoice?.customInstructions || "")
  });

  const prompt = SEGMENT_PROMPTS[segment](data);
  const result = await model.generateContent(prompt);
  
  return result.response.text();
}

export async function generateFullProposal(
  data: ProspectAnalysis,
  brandVoice?: BrandVoiceConfig
): Promise<ProposalContent> {
  const segments: Segment[] = ['hero', 'current_state', 'opportunities', 'roi', 'investment', 'next_steps'];
  const content: Partial<ProposalContent> = {};

  for (const segment of segments) {
    const text = await generateProposalSegment(segment, data, brandVoice);
    
    // Parse JSON segments, keep text segments as-is
    if (['hero', 'investment', 'next_steps'].includes(segment)) {
      content[segment] = JSON.parse(text);
    } else {
      content[segment] = text;
    }
  }

  return content as ProposalContent;
}
```

## Brand Voice Config

```typescript
interface BrandVoiceConfig {
  agencyId: string;
  
  // Tone
  formality: "formal" | "semi_formal"; // "Jūs" always, but warmth level varies
  enthusiasm: "confident" | "enthusiastic" | "understated";
  
  // Custom terminology
  customTerminology: Record<string, string>; // Override defaults
  
  // Forbidden phrases (agency-specific)
  forbiddenPhrases: string[];
  
  // Example proposals (for tone learning)
  exampleProposals: string[];
  
  // Custom system instructions
  customInstructions: string;
}
```
