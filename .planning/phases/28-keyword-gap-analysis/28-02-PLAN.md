# Task 28-02: Prospect Service Gap Analysis Integration

## Objective

Extend ProspectService to execute keyword gap analysis by discovering competitors and aggregating unique keywords. This task implements the business logic layer that orchestrates competitor discovery and domain intersection calls.

## Tasks

### 1. Add competitor discovery service method (TEST FIRST)
- [ ] Write test for `ProspectAnalysisService.discoverCompetitors()`
- [ ] Implement `discoverCompetitors(prospectId: string, limit?: number)`
  - Calls `dataforseoClient.prospect.competitorsDomain()`
  - Filters by relevance score (intersections >= 10)
  - Stores top N competitors on analysis record
- [ ] Test validates competitor filtering logic
- [ ] Test validates storage updates analysis record

### 2. Add keyword gap analysis service method (TEST FIRST)
- [ ] Write test for `ProspectAnalysisService.analyzeKeywordGaps()`
- [ ] Implement `analyzeKeywordGaps(analysisId: string, competitorDomains: string[])`
  - Loops through each competitor domain
  - Calls `dataforseoClient.prospect.domainIntersection()` for each
  - Aggregates unique keywords (dedupe by keyword string)
  - Calculates opportunity score for each gap
  - Stores sorted gaps on analysis record
- [ ] Test validates deduplication logic
- [ ] Test validates opportunity scoring integration
- [ ] Test validates cost aggregation across multiple calls

### 3. Add end-to-end gap analysis workflow (TEST FIRST)
- [ ] Write test for `ProspectAnalysisService.runGapAnalysis()`
- [ ] Implement `runGapAnalysis(prospectId: string)`
  - Step 1: Discover competitors (top 3)
  - Step 2: For each competitor, fetch keyword gaps
  - Step 3: Aggregate and score all gaps
  - Step 4: Update analysis record with results
  - Returns summary stats (total gaps, avg opportunity, etc.)
- [ ] Test validates full workflow with mocked DataForSEO calls
- [ ] Test validates error handling (no competitors found, API failures)

### 4. Add ProspectAnalysisService to server features (TEST FIRST)
- [ ] Write test for service instantiation and method availability
- [ ] Create `src/server/features/prospects/services/ProspectAnalysisService.ts`
- [ ] Export service from `src/server/features/prospects/index.ts`
- [ ] Test validates service integration with existing ProspectService
